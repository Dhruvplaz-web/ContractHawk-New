# 🦅 ContractHawk-New
**AI-Powered Legal Defense & Contract Neutralizer**

ContractHawk is a privacy-first legal auditor built for the **Innovate Maharashtra – Gemini GenAI Hackathon 2026**. It identifies predatory clauses in contracts and provides fair-market alternatives using Google's Gemini 1.5 Flash.

## 🚀 Core Functionalities
* **AI Audit Engine:** Scans PDF/DOCX files to find hidden legal risks.
* **The Neutralizer:** Automatically generates "fair" replacement text for risky clauses.
* **Legal Identity Calibration:** Adjusts audits based on jurisdiction (India, US, etc.) and negotiation posture.
* **The Kill Switch:** One-click permanent data purge for absolute user privacy.
* **Hacker Terminal UI:** A sophisticated dark-mode experience with live threat intelligence feeds.

## 🛠️ Tech Stack
* **Backend:** Python / Flask
* **AI:** Google Gemini GenAI API
* **Database:** SQLite (SQLAlchemy)
* **Frontend:** Tailwind CSS / JavaScript (Glassmorphism UI)
* **Extraction:** PyMuPDF / python-docx

## 🛠️ Installation & Setup
1. Clone the repo: \`git clone https://github.com/Dhruvplaz-web/ContractHawk-New.git\`
2. Install dependencies: \`pip install -r requirements.txt\`
3. Setup your API Key in a \`.env\` file: \`GEMINI_API_KEY=your_key_here\`
4. Run the app: \`python run.py\`
